# Bitrix-tinker

Порт `laravel/tinker`

# Установка

1. `composer require arrilot/bitrix-tinker`
2. Регистрируем команду `Arrilot\BitrixTinker\Console\TinkerCommand` в консольном приложении на базе symfony/console

# Использование

Запускаем `php our_console_app.php tinker`
